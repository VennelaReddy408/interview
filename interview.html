11.What is box-modal?

The box model in CSS describes how elements are structured and how their sizes are calculated in the layout of a webpage. Each element is considered a rectangular box, and the box model consists of several components.

12.What are the box-modal properties? (explain)

There are four properties?in box model they are
Content: This is the innermost part where text, images, and other media appear.
Padding: This is the space between the content and the border. It creates space inside the box, around the content. Padding is transparent and can be set individually for each side (top, right, bottom, left).
Border: This surrounds the padding (if any) and content. You can customize the border's width, style, and color.
Margin: This is the outermost space that creates distance between the element and other elements. Margins are also transparent and can be set for each side.

14.What is d/f b/w outline and border?

Border: The border is part of the element's box model and takes up space in the layout. It affects the dimensions of the element, increasing its width and height.
Outline: The outline does not take up space in the box model. It is drawn outside the border and does not affect the element's dimensions or layout.

35.What is a flex box in css?

36.What is a grid layout system?

A grid layout is a multi-dimensional design approach where content is organized into rows and columns

37.d/f b/w the flex vs grid?

Flexbox: Ideal for simpler, one-dimensional layouts where you need flexibility along a single axis (row or column).
Grid: Best for two-dimensional layouts where you need to control both rows and columns simultaneously and create more complex structures.

38.What are media queries in css? why do we need to use them?

Media Queries: Allow you to apply CSS styles conditionally, based on characteristics of the device (like width, height, resolution, or orientation).
Need for Media Queries: They enable responsive designs, improving user experience across a variety of devices, making web pages more adaptable, accessible, and user-friendly.

42.what is d.f b/w the transitions vs transforms vs animations?

Use Transitions: When you want to create smooth changes for simple interactions, like hover effects.
Use Transforms: When you want to change the shape or position of an element without needing to animate it.
Use Animations: When you need to create intricate animations that involve multiple steps or properties over time.
